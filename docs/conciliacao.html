Instruções de Conciliação

O que é Conciliação?

A conciliação é o processo de verificar se os registros internos do Instituto estão alinhados com os registros externos (como extratos bancários) ou com a realidade física (como o estoque de materiais). Este processo é fundamental para garantir a transparência, precisão e confiabilidade das prestações de contas.

Conciliação Financeira

Verifica se os lançamentos financeiros registrados no sistema (entradas e saídas) correspondem aos lançamentos do extrato bancário, garantindo que todo dinheiro que entrou ou saiu foi devidamente registrado e categorizado.

Conciliação de Bens

Verifica se os registros de entrada e saída de materiais (doações físicas, equipamentos, etc.) estão corretos, garantindo que todo material recebido foi devidamente registrado e que as distribuições correspondem ao que foi recebido.

Por que é importante?

  • Garante transparência na prestação de contas
  • Identifica erros, omissões ou fraudes
  • Assegura que todos os recursos foram utilizados conforme planejado
  • Facilita auditorias e fiscalizações
  • Melhora a gestão financeira e de recursos do Instituto

Como Realizar a Conciliação

Conciliação Financeira

  1. Importe os dados necessários:
    • Registros de entradas financeiras (doações recebidas)
    • Registros de saídas financeiras (pagamentos realizados)
    • Extrato bancário (exportado do seu banco)
  2. Analise o resultado da conciliação automática:
    • Conciliado: Lançamentos que possuem correspondência exata no extrato
    • Pendente: Lançamentos internos sem correspondência no extrato
    • Divergente: Lançamentos com valores diferentes entre sistema e extrato
    • Não identificado: Lançamentos no extrato sem registro interno
  3. Resolva as divergências:
    • Para lançamentos pendentes: verifique se o pagamento/recebimento foi realmente efetivado
    • Para lançamentos não identificados: registre-os no sistema interno
    • Para taxas bancárias: registre-as como despesas administrativas
    • Para divergências de valor: corrija o valor no sistema interno
  4. Exporte o relatório de conciliação:

    Após resolver todas as divergências, exporte o relatório final para documentação e prestação de contas.

Conciliação de Bens

  1. Importe os dados de estoque:
    • Registros de entradas de materiais (doações recebidas)
    • Registros de saídas de materiais (distribuições realizadas)
  2. Analise o resultado da conciliação automática:
    • Conciliado: Itens com entradas e saídas balanceadas
    • Pendente: Itens com mais entradas que saídas (estoque disponível)
    • Divergente: Itens com mais saídas que entradas (saldo negativo)
  3. Resolva as divergências:
    • Para itens pendentes: verifique se o estoque físico corresponde ao saldo disponível
    • Para itens divergentes: registre entradas faltantes ou corrija registros de saída
    • Realize contagem física para confirmar os saldos reais
  4. Exporte o relatório de estoque:

    Após resolver todas as divergências, exporte o relatório final para documentação e prestação de contas.

Exemplos Práticos

Exemplo de Conciliação Financeira

Cenário:

O Instituto registrou uma doação de R$ 5.000,00 via PIX no dia 10/05, mas o extrato mostra um crédito de R$ 5.000,00 apenas no dia 12/05.

Resultado da conciliação:

O sistema marcará como Conciliado, pois encontrou correspondência dentro da janela de ±3 dias, mas criará um gap de "diferença de data" para revisão.

Ação recomendada:

Verificar se a diferença de data é aceitável (compensação bancária normal) ou ajustar a data do lançamento interno para corresponder ao extrato.

Cenário:

O extrato mostra um débito de R$ 89,90 com descrição "TARIFA PACOTE MENSAL", mas não há lançamento correspondente no sistema.

Resultado da conciliação:

O sistema marcará como Não identificado e criará um gap de "taxa bancária".

Ação recomendada:

Registrar esta taxa como despesa administrativa na categoria "Taxas Bancárias".

Exemplo de Conciliação de Bens

Cenário:

O Instituto recebeu 100 camisetas como doação e registrou a distribuição de 60 camisetas em um evento.

Resultado da conciliação:

O sistema marcará como Pendente com saldo de 40 camisetas disponíveis.

Ação recomendada:

Verificar se as 40 camisetas estão fisicamente no estoque. Se não estiverem, registrar a saída correspondente.

Cenário:

O sistema mostra a distribuição de 50 kits escolares, mas só há registro de entrada de 30 kits.

Resultado da conciliação:

O sistema marcará como Divergente com saldo negativo de 20 kits.

Ação recomendada:

Registrar a entrada dos 20 kits faltantes ou verificar se houve erro no registro de saída.

Dicas e Melhores Práticas

Para Conciliação Financeira

  • Realize a conciliação pelo menos uma vez por mês
  • Mantenha descrições detalhadas nos lançamentos
  • Registre taxas bancárias assim que aparecerem no extrato
  • Utilize categorias consistentes para facilitar a análise
  • Documente justificativas para divergências não resolvidas
  • Mantenha comprovantes de todas as transações

Para Conciliação de Bens

  • Realize contagem física do estoque regularmente
  • Registre entradas e saídas no momento em que ocorrem
  • Padronize os nomes dos itens para evitar duplicidades
  • Documente a origem de todas as doações recebidas
  • Registre o destino de todas as distribuições
  • Fotografe itens de maior valor para documentação

Lembre-se

A conciliação não é apenas uma obrigação administrativa, mas uma ferramenta poderosa para garantir a transparência e eficiência na gestão dos recursos do Instituto. Uma conciliação bem feita facilita prestações de contas, auditorias e demonstra o compromisso da organização com a boa gestão dos recursos recebidos.
